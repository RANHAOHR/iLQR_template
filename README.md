# iLQR_template

The most basic iLQR template using a car dynamic without constraint & collision avoidance costs

## Dependances

**matplotlib:** 
```
	pip install matplotlib
```

### Build iLQR
```
	cd iLQR_template/build
	cmake ..
	sudo make install
```
**Compile & Run iLQR (In build directory)
```	
	cmake --build .
	./iLQR
```
